# Sealth – Digital Sexual Health Application

Sealth is a mobile-based digital sexual health application developed using Flutter as part of a software development coursework project. The application aims to improve access to sexual health information and services by providing users with educational resources, clinic information, appointment management, and location-based healthcare support through a user-friendly mobile interface.

---

## Features

### User Authentication

* User registration and login functionality
* Secure account management
* Personalized user experience

### Sexual Health Resources

* Access to educational sexual health information
* Browse articles and health-related content
* Improve awareness through reliable digital resources

### Clinic Directory

* View available healthcare clinics
* Access clinic information and contact details
* Browse nearby healthcare facilities

### Location-Based Services

* Integration with location services
* Discover nearby clinics based on user location
* Improve accessibility to healthcare providers

### Appointment Management

* Schedule healthcare appointments
* View upcoming appointments
* Manage appointment information within the application

### Database Integration

* Cloud-based data storage and synchronization
* Real-time data management
* Persistent storage of user and clinic information
